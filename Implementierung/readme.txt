Erstellt mit Apache NetBeans IDE 12.0
Übersetzt mit Java 1.8.0_251
Es wurden keine zusätzlichen Bibliotheken eingebunden.
Um Laufzeittests mit Instanzen bestimmter Größe durchführen zu können muss 
eventuell der als Standard verwendete Cache vergrößert werden, z.B. mit
"java -Xmx4096m -jar SearchTrees.jar" um maximal 4096MB zur Verfügung zu haben. 
SearchTrees.bat macht genau dies für Windows Systeme.
